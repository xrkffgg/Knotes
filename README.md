## 介绍

- 本项目以 [`Vuepress`](https://vuepress.vuejs.org/zh/) 为基础，用于记录各种 文章类

## 线上预览地址

- [地 址](https://xrkffgg.github.io/Knotes/)

## 启动

- `yarn start`

## 打包

- `yarn build`

> 有点坑的地方是，打包后，本地打开无主题，扔到 `GitHub.io` 正常。
> 
> `deploy.sh` 根据当前环境，仅供参考
